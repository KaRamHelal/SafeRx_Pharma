## Collection Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `base_url` | `https://saferx.online` | API base URL |
| `api_key` | *(empty)* | Your SafeRx API key — auto-saved after verification |

## Quick Start

1. **Import** — Open Postman → click **Import** → drag the downloaded JSON file
2. **Get API Key** — Run request **1. Request Verification Code** with your email
3. **Check Email** — Copy the 6-digit code from your inbox
4. **Verify** — Run request **2. Verify and Get API Key** with your email + code
5. **Done** — The API key is auto-saved to the collection. All other requests will use it automatically.

## Setup Details

### Step 1: Request a Free API Key

Open **1. Request Verification Code** and change the email:

```json
{
  "email": "your@email.com"
}
```

Send the request. Check your email for a 6-digit verification code.

### Step 2: Verify and Receive Key

Open **2. Verify and Get API Key** and enter your email + code:

```json
{
  "email": "your@email.com",
  "code": "123456"
}
```

Send the request. On success, the response includes your `api_key` — it's automatically saved to the collection variable by the post-response script.

### Step 3: Run Safety Checks

All remaining requests use `{{api_key}}` in the `X-SafeRx-API-Key` header. Just send them — no manual configuration needed.

## Example Scenarios

- **Basic Check:** Augmentin 1g + Marivan + Glucophage 500 — tests fuzzy matching and multi-drug interactions
- **Patient Profile:** Glucophage + Concor for elderly diabetic — triggers population and condition alerts
- **DDI Only:** Augmentin + Marivan with `include: ["ddi"]` — returns only interaction data
- **Arabic:** Augmentin + Antinal with `lang: "ar"` — all descriptions in Arabic
- **Pregnancy:** Marivan + Cataflam with `include: ["pllr"]` — Marivan triggers Black Box Warning

## Rate Limits (Free Tier)

- 20 requests/minute, 60 requests/day
